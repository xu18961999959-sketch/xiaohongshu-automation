🥇 **第一步：画出你的"能报圈"**

首先，筛选出你能报的所有岗位：

**硬性条件筛选：**
☑️ 学历符合
☑️ 专业对口（或不限）
☑️ 户籍符合
☑️ 年龄符合
☑️ 政治面貌符合
☑️ 基层工作经历符合
☑️ 其他条件符合

筛完之后，你会得到一个"可报岗位池"。

🥈 **第二步：计算"竞争指数"**

对每个可报岗位，计算竞争指数：

**竞争指数 = 预估报名人数 ÷ 招录人数**

如何预估报名人数？

**方法一：参考往年数据**
同岗位或类似岗位的历史报名人数

**方法二：条件推算法**
限制条件越多，报名人数越少
- 限专业：减少60%竞争者
- 限户籍：减少70%竞争者
- 限应届：减少50%竞争者

**方法三：实时数据跟踪**
报名期间关注官方公布的报名人数

🥉 **第三步：匹配"个人优势"**

最后一步，选择能发挥你优势的岗位：

**优势匹配清单：**

| 你的优势 | 适合岗位 |
|---------|---------|
| 专业对口 | 限专业岗位 |
| 盐城户籍 | 限户籍岗位 |
| 应届身份 | 限应届岗位 |
| 分数高 | 竞争大但好的岗位 |
| 分数一般 | 竞争小的岗位 |
| 面试能力强 | 面试占比高的岗位 |

💎 **黄金法则总结**

**最优选择 = 能报 + 竞争小 + 发挥优势**

三个条件同时满足的岗位，就是你的最佳选择！

⭐ **实战案例**

学员小王，普通二本，会计专业，盐城本地人。

按这个方法：
- 第一步：筛出可报岗位32个
- 第二步：计算竞争指数，排除高于50:1的
- 第三步：重点看限盐城户籍+限会计专业的

最终选择：XX县财政局，竞争比28:1，成功上岸！

💣 **雷区一：只看单位名字，不看实际待遇**

很多人一看"XX局"就觉得高大上，冲！

实际上：
- 同一个系统，不同层级差距很大
- 有些单位名字响亮，待遇一般
- 有些单位默默无闻，待遇很好

**正确做法**：多方了解实际情况

💣 **雷区二：盲目追求市区，放弃县区机会**

"我一定要留在市区！"

结果：
- 市区竞争激烈，考不上
- 县区机会放弃，错过上岸
- 年年陪考，年年落榜

**正确做法**：先上岸，再说发展

💣 **雷区三：忽视限制条件的价值**

很多人觉得"不限专业"选择多，机会大。

实际上：
- 不限专业 = 所有人都能报 = 竞争最大
- 限专业限户籍 = 竞争对手少一大半

**正确做法**：主动选限制条件多的岗位

💣 **雷区四：只看招录人数，不看报名人数**

"这个岗位招5个人，机会大！"

实际上：
- 招5人可能报500人（100:1）
- 招1人可能报20人（20:1）

**正确做法**：关注竞争比，不是招录数

💣 **雷区五：报名最后一天才选**

"我再想想..."

结果：
- 好岗位已经报满
- 没时间做充分分析
- 慌乱中随便选一个

**正确做法**：提前准备，中期报名

⭐ **正确选岗姿势**

1️⃣ 岗位表出来前就准备好条件
2️⃣ 第一时间筛选可报岗位
3️⃣ 分析竞争情况再做决定
4️⃣ 报名中期（第3-4天）提交
5️⃣ 持续关注报名人数变化

选岗最核心的数据是什么？**竞争比！**

今天教你怎么算、怎么用。

📐 **竞争比基础公式**

```
竞争比 = 报名人数 ÷ 招录人数
```

举例：
- 岗位招2人，报名100人
- 竞争比 = 100÷2 = 50:1

📊 **盐城竞争比参考区间**

| 竞争比 | 难度评价 | 建议 |
|--------|----------|------|
| <20:1 | 低竞争 | 优先选择 |
| 20-50:1 | 中等竞争 | 可以考虑 |
| 50-100:1 | 高竞争 | 需要实力 |
| >100:1 | 神仙打架 | 谨慎选择 |

🔮 **报名前如何预估竞争比？**

**预估公式**

```
预估报名数 = 往年同岗位报名数 × 今年系数
```

今年系数根据：
- 整体报名趋势（今年比去年+25%）
- 岗位条件变化
- 热度变化

**快速预估法**

根据限制条件判断：

| 岗位类型 | 预估竞争比 |
|----------|-----------|
| 三不限市直 | 100:1以上 |
| 三不限县区 | 50-100:1 |
| 限专业 | 30-50:1 |
| 限专业+限户籍 | 15-30:1 |
| 限专业+限户籍+限应届 | <15:1 |

📈 **报名期间实时跟踪**

报名期间官方会公布报名人数：

- 前3天数据仅供参考
- 最后1天会有大量报名
- 最终数据可能是中期的2-3倍

💡 **竞争比决策原则**

**原则一：同等条件选竞争比小的**
待遇差不多，选竞争小的更划算

**原则二：竞争比不是唯一因素**
还要考虑岗位质量、发展前景

**原则三：给自己留余量**
分数一般的，选低竞争比更稳妥

什么是"捡漏上岸"？

就是用相对低的分数，考上相对好的岗位！

🔍 **为什么会有"漏"可捡？**

**原因一：信息差**
很多人不知道这个岗位/不了解实际情况

**原因二：偏见**
觉得某些单位/地区不好，不愿意报

**原因三：条件限制**
限制条件多，很多人报不了

**原因四：时间差**
报名最后才发现，来不及准备

📍 **盐城哪里容易"捡漏"？**

**地区维度**
- 响水、滨海等较偏县区
- 各县区的乡镇岗位
- 经济一般但待遇还可以的地方

**单位维度**
- 不太"高大上"的单位
- 冷门但稳定的部门
- 新设或调整的岗位

**条件维度**
- 限盐城户籍的岗位
- 限冷门专业的岗位
- 限男性/限女性的岗位
- 有工作经历要求的岗位

🎯 **找"漏"的方法**

**方法一：逆向思维**
别人不想去的地方，可能正是你的机会

**方法二：数据分析**
对比往年数据，找出低竞争岗位的规律

**方法三：及时跟踪**
报名期间关注报名人数，发现冷门岗位

**方法四：用好限制条件**
你有的条件，正是筛掉竞争对手的工具

⚠️ **捡漏的注意事项**

1️⃣ 捡漏不是捡差的，而是捡适合的
2️⃣ 要确认岗位实际情况再报
3️⃣ 低竞争不等于没难度
4️⃣ 不要只为捡漏而放弃好机会

💡 **张老师的建议**

先确定自己能接受的底线，在底线之上找竞争最小的。

不要为了捡漏去一个你根本不想去的地方！
