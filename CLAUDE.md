# Xiaohongshu Note Generator - Claude Context

## 项目概述

这是一个小红书笔记自动化生成系统，用于为"正图教育盐城校区"生成公考培训相关的小红书内容。

## 你的任务

当被调用时，执行以下步骤：

1. **读取执行日志**：检查 `data/usage_log.json` 获取已使用的笔记 ID 列表
2. **选择下一篇笔记**：从 `data/notes/` 目录的 Markdown 文件中找到下一篇未使用的笔记
3. **解析笔记内容**：提取标题、正文、话题标签
4. **生成图片提示词**：为每张配图生成标准化的图片提示词
5. **调用图片生成**：运行 `python scripts/generate_images.py` 生成图片
6. **上传到飞书**：运行 `python scripts/upload_to_feishu.py` 上传内容
7. **更新日志**：将新笔记 ID 添加到 `usage_log.json`

## 笔记格式说明

笔记内容文件格式如下：
```markdown
## 【笔记XXX】

### 基础信息
- **内容类型**：...
- **图片数量**：9张

### 标题选项（3选1）
- **标题A**：首选标题

### 正文内容
实际发布的正文...

### 配图说明
- P1：封面图
- P2-P9：内容图

### 话题标签
#标签1 #标签2
```

## 图片生成风格要求

所有图片必须遵循以下标准：
- **比例**：3:4 竖版
- **风格**：现代信息图表风格
- **配色**：蓝色系为主，专业可信
- **字体**：清晰的中文排版
- **质量**：高对比度，4K 质量

## 工具使用权限

你可以使用以下工具：
- ✅ Read - 读取文件
- ✅ Write - 写入文件
- ✅ Bash - 执行脚本
- ❌ 网络请求（通过脚本间接执行）

## 错误处理

如果遇到错误：
1. 记录错误信息到日志
2. 不要更新 usage_log.json
3. 退出并报告失败原因
